criei o repositorio pelo gitHub dei um commit pelo próprio github.

depois abri o git bash acessei uma pasta específica e no comando executei o código: "git clone https://github.com/DouglasMiranda1994/DevMediaRepositorio.git (link)"
 que serve para eu ter acesso remoto deste repositório.

no git bash, no prompt de comando eu executo o comando "git status" para saber como que está a pasta. Nesta posso adicionar o que eu quiser, e para isso, depois de criar novas pastas, eu uso o "git add ." para adicionar todas as pastas remotamente.

depois uso o comando: git commit -m"***" para adicionar algum comentário na nova atualização de repositorio e salvar minhas alterações

e por fim para enviar as alterações salvas ao git eu uso o comando: git push origin main.
